# JobLogger

## 概要
就職活動の動きを登録、確認できるアプリです。

## 開発背景
就活サイトでは各企業のインターン情報が
サイトごとに分散しており一元管理が困難なため自作しました。

## 使用技術
- フロントエンド：HTML / CSS / JavaScript
- バックエンド：Firebase
- バージョン管理：Git / GitHub

## 機能一覧
- インターンシップや合同説明会など予定されてる内容を登録
- 登録内容を表形式で表示(日付で昇順でソート)
- 登録内容を削除
